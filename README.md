# Lokum Ops Website

Modern, responsive website for LokumOps - Cloud & AI Solutions, AWS Partner.

## Features

- Rotating hero text with dynamic subtitles
- Sequential reveal animations for expertise cards
- Expandable solution cards
- Smooth scroll navigation
- Fully responsive design
- Clean, modern aesthetic inspired by Roots design

## Structure

- `index.html` - Main website page
- `styles.css` - All styling and animations
- `script.js` - Interactive features and animations
- `privacy.html` - Privacy policy page

## Assets Needed

Place your assets in an `assets` folder:
- Logo: `assets/logo.png`
- Client logos (6 logos)
- App screenshots/images (optional)

## Customization

Update the rotating text, colors, or content directly in the HTML and CSS files. The color scheme is defined in CSS variables at the top of `styles.css`.

## Deployment

Simply upload all files to your web hosting service. No build process required.
